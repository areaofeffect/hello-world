State Machine Game - Challenge

How to Play:

  • SPACE: Start game / Return to menu
  • CLICK: Score points during gameplay
  • You have 5 seconds to click as many times as possible!
  • Watch the game state change: MENU → PLAYING → GAMEOVER

Explore the Code:

Notice how each state has its own draw function and behavior.

Try modifying: Change the timer duration, add new states (PAUSED, INSTRUCTIONS), or create your own game using this pattern!
